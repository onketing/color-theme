# 온케팅 컬러 테마 쇼케이스 (color-theme)

홈페이지 제작 외주를 받을 때, 클라이언트에게 색감(컬러 테마)을 제안하고 미리 보여주기 위한 쇼케이스 웹앱입니다.

- **현재 의뢰업체:** 초이스 행정사 사무소
- **요청 사항:** [로어스(lawus.co.kr)](https://lawus.co.kr/) 와 동일한 색감
- **결과물:** 로어스 홈페이지 CSS에서 실측 추출한 **시그니처 테마 1종 + 동일 계열 변형 5종 = 총 6종**

## 색감 추출 근거

로어스 홈페이지(`/resource/css/style.css`)의 CSS 커스텀 프로퍼티에서 실제 브랜드 색상을 추출했습니다.

| 역할 | 추출 값 | 비고 |
| --- | --- | --- |
| 주조색 (brand-600) | `#7a6955` | 따뜻한 토프 브라운 |
| 강조색 (accent-base) | `#e76f51` | 테라코타 |
| 강조색 (light/dark) | `#f0a794` / `#a23216` | |
| 본문 텍스트 (text-base) | `#2c3e50` | 슬레이트 |
| 제목 (heading-base) | `#212121` | |
| 보조 텍스트 (text-light) | `#476481` | 스틸 블루 |
| 스톤 네이비 (stone) | `#00142d` / `#00367a` | |
| 샌드 베이지 | `#ded0b9` | |

## 제안 테마 6종

1. **로어스 시그니처** — 원본 추출 (토프 브라운 + 테라코타)
2. **테라코타 웜** — 더 포근한 클레이 톤
3. **딥 스톤 네이비** — 묵직한 네이비 + 샌드 브론즈
4. **샌드 & 토프** — 은은한 베이지 중심의 우아함
5. **스틸 블루 그레이** — 차분한 블루그레이 + 테라코타 포인트
6. **클레이 어스** — 깊은 커피 브라운 + 브릭 레드

각 테마는 컬러 팔레트(클릭 시 HEX 복사)와, 실제 "초이스 행정사 사무소" 홈페이지 화면에 적용된 라이브 미리보기를 함께 제공합니다.

## 기술 스택

- React 19
- Vite 6
- Tailwind CSS 4 (`@tailwindcss/vite`)
- Biome (lint + format)

## 실행

```bash
npm install      # 의존성 설치
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드 → dist/
npm run preview  # 빌드 결과 미리보기
npm run lint     # Biome 검사
npm run format   # Biome 포맷
```

## 새 의뢰업체 추가 방법

`src/data/themes.js` 의 `themes` 배열에 팔레트를 추가하면 자동으로 탭/비교 그리드에 반영됩니다. 의뢰업체가 바뀌면 `HomepagePreview.jsx` 의 목업 텍스트만 교체하면 됩니다.
